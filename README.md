## Project Overview

This is Poke Parse a lib or utility that you can use to get all the data you neeed to write your on Pokemon game 
This lib/utility uses https://pokemondb.net to update the data

## Features

List all Pokemon with BaseStats Evs Pokedex Entrys Evolutions Forms
List all moves in Pokemon with all values
List all Item in Pokemon with description


## Installation

cargo add poke-parse or git clone (standalone)
